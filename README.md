# Vet_clinic_business_process

Данный репозиторий предоставляет информацию о проектировании бизнесс-процесса <<Запись в ветеринарную клинику>>, создании структуры данных, проходящей по данному процессу, и представлению данной структуры через json-схему. Помимо этого в работе были изучены инстурменты работы с MongoDB.

1. Отчет о выполненном задании.pdf - содержит краткое описание реализованных заданий.
2. ShemBP.pdf - BPMN-диаграмма бизнесс-процесса записи для двух ролей (Хозяин собаки, Администратор вет.клиники)
3. table2.png - структура <<Запись на прием>>, поля которой заполняются при прохождении по этапам бизнесс-процесса
